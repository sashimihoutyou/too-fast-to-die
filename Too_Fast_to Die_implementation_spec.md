### S1-5. 戦闘ステートマシン

```
CombatState:
  INIT           → 戦闘開始演出 → ENVIRONMENT
  ENVIRONMENT    → 環境効果適用 → DRAW
  DRAW           → 5枚ドロー → INTENT
  INTENT         → 敵インテント表示 → PLAYER_TURN
  PLAYER_TURN    → プレイヤー行動（カード使用/逃走/ターン終了）→ DISCARD
  DISCARD        → 手札を捨て札へ → ENEMY_TURN
  ENEMY_TURN     → 敵行動実行 → CHECK_END
  CHECK_END      → 勝利/敗北/継続判定 → DRAW or VICTORY or DEFEAT
  VICTORY        → 報酬選択 → EXIT
  DEFEAT         → ゲームオーバー遷移 → EXIT
  FLED           → 逃走処理（燃料消費）→ EXIT
```

| エリート戦闘の想定ターン数 | 8〜12 |
| ボス戦闘の想定ターン数 | 12〜20 |

### S2-2. AP対効果基準（1APあたりの期待値）

| レアリティ | ダメージ期待値 | ブロック期待値 |
|---|---|---|
| コモン | 5〜7 | 5〜6 |
| アンコモン | 7〜9 | 7〜8 |
| レア | 9〜12 | 10〜12 |

燃料1消費ごとに追加ダメージ+4〜5を上乗せ。

### S2-3. ダメージ計算式

```
最終ダメージ = max(0, 基礎ダメージ + バフ補正 - 萎縮)

被ダメージ処理:
  if ブロック >= ダメージ:
    ブロック -= ダメージ
  else:
    残ダメージ = ダメージ - ブロック
    ブロック = 0
    HP -= 残ダメージ
```

### S2-4. 区間別スケーリング

| 区間 | 通常敵HP | 通常敵攻撃力 | エリートHP | エリート攻撃力 | ボスHP | ボス攻撃力 |
|---|---|---|---|---|---|---|
| 1 | 25〜40 | 6〜10 | 55〜70 | 10〜14 | 150 | 12〜25 |
| 2 | 35〜55 | 8〜12 | 70〜90 | 12〜18 | 200 | 14〜28 |
| 3 | 45〜65 | 10〜14 | 80〜100 | 14〜20 | 230 | 16〜30 |
| 4 | 50〜70 | 12〜16 | 90〜110 | 16〜24 | 260 | 18〜32 |
| 5 | 45〜65 | 10〜14 | 80〜110 | 14〜22 | 200〜300 | 16〜30 |

区間5は区間4より若干緩い通常敵＋キャラ固有ボスで難易度にメリハリをつける。

## S11. アート方針

### ビジュアルスタイル

**方向性: ダークなコミック調ピクセルアート（解像度高め）**

- 参考: Dead Cells、Darkest Dungeon のトーン。明るすぎず暗すぎない
- カード: 64×64〜128×128のイラスト。手描き風タッチ。枠はレアリティで色分け

## S15. 未決事項の解決（企画設計書v2 第26章対応）

### コカトリスが元レイダーを追い続ける理由

元レイダーはコカトリスの秘密拠点の位置と奴隷輸送ルートの情報を知っている。追放時にその情報の価値に気づかなかったが、チェインリンクとの取引が拡大するにつれ情報漏洩のリスクが顕在化。さらに、殺害命令を仕組んだ副首領が「追放ではなく殺害すべきだった」と主張し、面子の回復も兼ねて追手を出している。

### コカトリスのボス造形

**副首領「レイヴン」** — 元レイダーがかつて姉のように慕っていた女性。冷酷だが合理的で、排除工作の黒幕。元レイダーを追放した本当の理由は「自分の地位を脅かす存在になりつつあった」から。少年殺害命令は排除の口実。ボス戦で真実を突きつけてくる。

### カード強化のキャラ固有効果

S3の各カードテーブルの「強化後」列に記載済み。原則:
- 攻撃系: ダメージ+50%
- 防御系: ブロック+50%
- キャラ固有: カードごとに個別設計（係数変更、追加効果等）

### 各キャラがNPCとして登場する条件

| プレイキャラ | NPC登場するキャラ | 条件 | 内容 |
|---|---|---|---|
| 放浪者 | 調教師 | 区間4以降、チェインリンク情報入手済 | NPCとして出会い会話。娘だと確信するイベント |
| 調教師 | 放浪者 | 区間3以降、ランダム | NPCとして出会い、父の友人と知らず手助けを受ける |
| 任意 | カルティスト | V8カルト関連イベント | 名もなきV8信者として背景に登場 |
| 任意 | 覇者 | 区間4〜5、低確率 | 道端で獣を素手で倒す大男を目撃するのみ |
| 任意 | 元レイダー | コカトリス遭遇イベント | レイダーの話題で名前が出る程度 |

### 覇者の旅人の妹の戦闘リスク

旅人の妹はHP20の保護対象として戦闘に存在する。敵は確率で旅人の妹を狙う（30%）。被弾すると覇者の闘気+20（怒りの反応）。HP0で死亡した場合、以降のエンディング分岐が変化し、覇者固有の喪失イベントが発生する。

### 調教師の人間同行者へのスタンス

人間同行者の加入は可能だが、専用テキストで「虎が警戒している」旨が表示される。人間同行者がいる間、虎の肩代わり確率が30%低下（虎が同行者を信用していない表現）。犬は例外。

### 汚染地帯の報酬テーブル

| 報酬 | 出現確率 | 内容 |
|---|---|---|
| 旧世界の医薬品 | 20% | 医薬品×2 |
| 汚染された燃料 | 30% | 燃料+5, 汚染1蓄積 |
| テクノロジー部品 | 15% | レアバイクパーツ |
| ノウアスフィアの残留データ | 10% | オアシス情報（ランダムカテゴリ/段階2） |
| 旧世界の武器 | 15% | レアカード |
| 空っぽ | 10% | 何もなし（汚染リスクのみ） |
