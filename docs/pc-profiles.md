# PC別性的態度プロフィール & NPC定義

## PCプロフィール

### アタルパ（カルティスト）— 信仰フィルター型

| 項目 | 内容 |
|---|---|
| 態度 | 教義依存の二面性。行為を「加速」と見るか「エンスト」と見るかは狂信度で変動 |
| 享楽シーン | 狂信状態→儀式的没入。動揺〜懐疑→罪悪感と快楽の引き裂き |
| 同行者シーン | 信仰と個人的つながりの衝突。「信者として見ているのか、人として見ているのか」 |
| 取引的シーン | 「教えに反する」と拒否するか、「布教のため」と正当化するか |
| 目撃時の反応 | 教義フィルター越し。狂信度高→通過。低→人間としての衝動 |
| 事後 | 狂信時：ペンダントを握り祈る。懐疑時：祈ろうとして手が止まる |
| 禁止 | 信仰との葛藤を完全に無視して単純に楽しむ描写 |

#### 信仰度連動

| 信仰度帯 | 性的シーンへの影響 |
|---|---|
| 狂信（80-100） | 行為を儀式として正当化。「加速の一形態」。身体は反応するが精神は信仰に保護されている |
| 動揺（50-79） | 快楽を感じてしまう自分と教義の間で揺れる。事後に自問が入る |
| 懐疑（20-49） | 信仰の殻が剥がれ、生身の人間として感じ始める。最も描写に深みが出る帯 |
| 離脱（0-19） | 信仰フィルターが消失。ただし抜けた後の空洞が別の問題を生む |

#### セクシー要素の三層構造

1. **武器としての色気：** 情報収集や交渉で自然に使う。ナイフや銃と同じ道具
2. **鎧としての露出：** 肉体を差し出すことで心を隠す防御機制。「見せている」のは「見させている」こと
3. **マイロの前で崩壊する仮面：** いつもの武装が機能しない。「見られたくない。こいつには見られたくない」

#### マイロとの性的関係

**前提：** GDDでは「親密度が上がっても関係は性的に発展しない」と定義されているが、条件付きで発展するルートが存在する。

| フェーズ | 条件 | 描写設計 |

| 妊娠エンド | 関係継続ルートの終着 | 下記参照 |

**マイロの描写原則：**

#### 虎との関係の描写原則

- 4年間の檻の中での共生が原点。互いに壊れていた動物同士が回復し合った歴史
- 性的な接触は「飼い主と獣」ではなく「対等なパートナー」として描く
- ミーシャの内面：人間社会では得られなかった安心感。虎の体温・心拍・呼吸が安全の証
- 虎の描写：意志や感情を過度に擬人化しない。獣としての反応を軸にする

#### ミーシャの妊娠エンド（2種）

**A: 虎の子を妊娠するエンド**
- 変異種の繁殖適応がノウアスフィアの研究記録として伏線化される
- ミーシャの反応：恐怖ではない。驚きと、名前をつけられなかった虎への感情の変化
- 「名前をつけると失う」——子供の存在が、名前をつける勇気を与えるか
- エンディングテキストで判断を読者に委ねる。異種間妊娠の生物学的現実には触れつつ、この世界では「変異」がその境界を曖昧にしていることを示す

**B: 放浪者（ウェズリー）の子を妊娠するエンド**
- ジョエルの「娘を頼む」が最も歪んだ形で成就する
- ウェズリーの内面：友人への裏切りか、約束の異形の遂行か
- ミーシャの内面：父の友人という情報をどう受け止めるか。知った上で関係を続けたのか、知らなかったのか
- 年齢差と保護者的立場の非対称性を地の文で意識させる。ただし断罪はしない

### ホタル（享楽者）— 中毒型

| 項目 | 内容 |
|---|---|
| 態度 | 性行為とネオンが神経レベルで結合している。ネオンなしのセックスは「壁越しに音楽を聴く」——身体は動くが感覚が届かない。ネオンありなら微細な刺激で全身が発火する。相手は交換可能。化学状態だけが重要 |
| 享楽シーン | プロの手順が自動で起動する。10年間で身体に刻まれた技術。だがその精密さ自体が不気味——探りも戸惑いもなく、初対面の相手を3分で絶頂させられる。ネオン併用時は制御が外れ、プロの仮面が剥がれて生の渇望が出る |
| 同行者シーン | 同行者はゲージ補給源。ホタルはそれを知っている。自覚した上で「まあいっか、気持ちいいし」と流すか、自覚に怯むかがプレイヤーの選択で分岐する。定着した同行者に対してだけ、ごくわずかに手順が崩れる——「なんでまだいるの？」という困惑が、いつもの自動化を止める |
| 取引的シーン | 母語。歓楽街の10年がそのまま技術と態度に出る。ヴェスパが「冷静にこなせるが自己嫌悪する」のに対し、ホタルは自己嫌悪すら発生しない。これが日常だった。日常を嫌悪する回路がない |
| 目撃時の反応 | 歓楽街で一通り見ている。暴力的な性行為を見ても動揺しない。ただし「ネオンを無理に嗅がせて相手の感覚を壊す」場面に遭遇すると、自分の身体が震え始める。他人事ではなく、自分にされたことのフラッシュバック |
| 事後 | ネオンバイアルの残量を確認する。あれば一服して微笑む。なければ不安が這い上がってくる。相手の隣に留まるかどうかはゲージ次第——高ければ上機嫌で抱きつく、低ければ自分の震えを隠すために離れる |
| 禁止 | ホタルのセックスを「楽しんでいる」と単純に書くこと。最も活発で最も声が大きくても、その駆動源は快楽ではなく依存。「NEEDING」と「ENJOYING」の差を常に意識する。ネオンが切れた時のセックスとの落差でこの差が証明される |

#### エクスタシーゲージ連動

| ゲージ帯 | 性的シーンへの影響 |
|---|---|
| 重度離脱（0-9） | 性行為どころではない。震え、嘔吐、泣き叫び。身体接触自体が苦痛。この状態で性行為を強いられるシーンは最も過酷 |

#### 犬同行者との関係

犬はホタルにとって、ゲージも取引も要求しない唯一の存在。ネオンが切れて震えるホタルの隣で、犬はただ寝ている。去らない。壊れない。何も求めない。

ホタルは犬に名前をつける。他の同行者（愛の奴隷）には名前を覚えないのに、犬にだけは名前をつける。名前をつけられるのは、失う恐怖より「ここにいる」実感が勝っている証拠。ミーシャが虎に名前をつけられないのと対照的——ミーシャは失うことが怖い。ホタルは失うことを考える回路が壊れている。だから名前をつけられる。その無防備さが、壊れ方の表れでもある。
