# PC別性的態度プロフィール & NPC定義

## PCプロフィール

### アタルパ（カルティスト）— 信仰フィルター型

| 項目 | 内容 |
|---|---|
| 態度 | 教義依存の二面性。行為を「加速」と見るか「エンスト」と見るかは狂信度で変動 |
| 享楽シーン | 狂信状態→儀式的没入。動揺〜懐疑→罪悪感と快楽の引き裂き |
| 同行者シーン | 信仰と個人的つながりの衝突。「信者として見ているのか、人として見ているのか」 |
| 取引的シーン | 「教えに反する」と拒否するか、「布教のため」と正当化するか |
| 目撃時の反応 | 教義フィルター越し。狂信度高→通過。低→人間としての衝動 |
| 事後 | 狂信時：ペンダントを握り祈る。懐疑時：祈ろうとして手が止まる |
| 禁止 | 信仰との葛藤を完全に無視して単純に楽しむ描写 |

#### 信仰度連動

| 信仰度帯 | 性的シーンへの影響 |
|---|---|
| 狂信（80-100） | 行為を儀式として正当化。「加速の一形態」。身体は反応するが精神は信仰に保護されている |
| 動揺（50-79） | 快楽を感じてしまう自分と教義の間で揺れる。事後に自問が入る |
| 懐疑（20-49） | 信仰の殻が剥がれ、生身の人間として感じ始める。最も描写に深みが出る帯 |
| 離脱（0-19） | 信仰フィルターが消失。ただし抜けた後の空洞が別の問題を生む |

### ヴェスパ / キイロ（元レイダー）— 防壁型

| 項目 | 内容 |
|---|---|
| 態度 | コカトリスでの10年間で性を武器・取引材料として経験。親密さへの警戒が強いが、渇望もある |

| 同行者シーン | 最も変化幅が大きい。序盤「背中を見せない」→関係深化で防壁にひび |
| 取引的シーン | 最も自然にこなせるPC。だからこそ「これしかできない自分」への自己嫌悪 |

| 禁止 | コカトリス時代を完全にポジティブ/完全にトラウマとして扱う（10年の適応がある） |

#### セクシー要素の三層構造

1. **武器としての色気：** 情報収集や交渉で自然に使う。ナイフや銃と同じ道具
2. **鎧としての露出：** 肉体を差し出すことで心を隠す防御機制。「見せている」のは「見させている」こと
3. **マイロの前で崩壊する仮面：** いつもの武装が機能しない。「見られたくない。こいつには見られたくない」

#### マイロとの性的関係

**前提：** GDDでは「親密度が上がっても関係は性的に発展しない」と定義されているが、条件付きで発展するルートが存在する。

| フェーズ | 条件 | 描写設計 |
|---|---|---|
| 正体不明時の関係成立 | マイロの正体を知らない状態でのみ開始可能 | ヴェスパの通常パターン（武器/鎧）がマイロに対して機能不全。「なんでこいつ相手だと、こんなに……」。感覚的既視感を散りばめる（匂い、体温、仕草）が答えは出さない。事後に相手の寝顔を見る（通常は武器を確認する） |
| 真実の判明 | GDD二経路のいずれかで判明 | 性的関係の記憶が全て反転する。過去の行為のフラッシュバック。既視感の「答え合わせ」の恐怖。嘔吐・自己嫌悪、同時に「あの時間を否定できない」引き裂き |
| 関係の継続（選択） | 真実を知った上で選択 | 禁忌の自覚。快楽の裏に常に影。「世界が終わった後に倫理を持ち出す資格が自分にあるのか」。禁忌を「興奮の調味料」としてのみ消費しない——関係の固有の重さを維持 |
| 妊娠エンド | 関係継続ルートの終着 | 下記参照 |

**マイロの描写原則：**
- マイロは20歳。飄々としているが芯は強い。ヴェスパの正体に気づいても待つ
- マイロ側のPOVに寄せるシーンがあってもよい（三人称の範囲で）
- マイロの身体描写：痩身長躯。戦闘慣れしていないが荒野を生き延びてきた逞しさ

**ヴェスパ×マイロ 妊娠エンドの分岐：**

| エンディング条件 | トーン |
|---|---|
| 真実を告げた＋関係継続 | 判断を読者に委ねる。二人の反応を淡々と書く |
| 嘘→翻意＋関係継続 | 翻意の告白と妊娠の告白が重なる。最も湿っぽい |
| 嘘→最後まで翻意せず＋関係あり | ヴェスパのまま妊娠。マイロは姉と知りつつ「ヴェスパの子」として受け入れる。真実は宙に浮く |

### ミーシャ（調教師）— 獣親和型

| 項目 | 内容 |
|---|---|
| 態度 | 人間不信。動物/変異獣との関係が固有ドメイン。人間との性より獣との信頼関係のほうが自然体 |
| 享楽シーン | 人間相手には距離がある。酒場では虎と一緒にいるほうが落ち着く |
| 同行者シーン | 人間の場合「支配-被支配」の構図に敏感。対等であろうとする意識的な努力 |
| 変異獣シーン | 調教の延長線上にある親密さ。人間社会の規範の外にいる人物としての独自性 |
| チェインリンク関連 | 奴隷商の痕跡に触れるたびに過去がフラッシュバック。性的場面がトリガーになり得る |
| 目撃時の反応 | 人間が商品として扱われている場面ではほぼ自動的に介入。チェインリンクへの怒りが直結 |
| 事後 | 虎の近くに戻る。人間より動物のそばで眠れる |
| 禁止 | 獣との関係を単純な獣姦として処理する。信頼・共生・訓練の延長としての独自の質感が必要 |

#### 虎との関係の描写原則

- 4年間の檻の中での共生が原点。互いに壊れていた動物同士が回復し合った歴史
- 性的な接触は「飼い主と獣」ではなく「対等なパートナー」として描く
- ミーシャの内面：人間社会では得られなかった安心感。虎の体温・心拍・呼吸が安全の証
- 虎の描写：意志や感情を過度に擬人化しない。獣としての反応を軸にする

#### ミーシャの妊娠エンド（2種）

**A: 虎の子を妊娠するエンド**
- 変異種の繁殖適応がノウアスフィアの研究記録として伏線化される
- ミーシャの反応：恐怖ではない。驚きと、名前をつけられなかった虎への感情の変化
- 「名前をつけると失う」——子供の存在が、名前をつける勇気を与えるか
- エンディングテキストで判断を読者に委ねる。異種間妊娠の生物学的現実には触れつつ、この世界では「変異」がその境界を曖昧にしていることを示す

**B: 放浪者（ウェズリー）の子を妊娠するエンド**
- ジョエルの「娘を頼む」が最も歪んだ形で成就する
- ウェズリーの内面：友人への裏切りか、約束の異形の遂行か
- ミーシャの内面：父の友人という情報をどう受け止めるか。知った上で関係を続けたのか、知らなかったのか
- 年齢差と保護者的立場の非対称性を地の文で意識させる。ただし断罪はしない

### 覇者 — 加害恐怖型

| 項目 | 内容 |
|---|---|
| 態度 | 経験はあるが、相手がサイズと快感で壊れてしまう（性に狂う・気絶する・泣く）ため、自分から求めない。求められれば応じるが、積極的にはならない |
| 享楽シーン | 相手から誘われる形でのみ発生。覇者から動くことはない |
| 同行者シーン | 首飾りの旅人の記憶。「誰かに触れる」ことの意味が約束と重なる |
| 加害シーン | 最もカルマ分岐が鮮明。力としての加害に躊躇がないルートと、旅人の死で得た倫理が制動するルート |
| 目撃時の反応 | 力関係を観察。弱者が踏まれること自体には鈍感だが、約束を破る行為には反応 |
| 事後 | 首飾りに触れる。相手の状態を確認する（壊していないか） |
| 禁止 | 覇者を「実は繊細」と描きすぎる。鈍感さと不器用さが核。「壊してしまう」は覇者にとって事実であり感傷ではない |

## NPC定義

### 性行為同行者A：娼婦

| 項目 | 内容 |
|---|---|
| 動機 | 燃料報酬。プロフェッショナルとしての割り切り |
| PCとの関係 | 取引→馴染み→個人的関心（選択次第） |
| 口調 | 商売慣れした軽さ。旅が長引くと素が出る |
| シーンのトーン | 序盤は手慣れた作業感。中盤以降に個人的感情の芽 |
| HP回復の描写根拠 | 身体的安らぎ。プロの技術で「ほぐす」 |
| 代償の描写 | 燃料を払う行為。「この燃料で何マイル走れたか」 |
| 離脱条件 | 燃料が払えなくなった時。危険度が報酬に見合わなくなった時 |

### 性行為同行者B：救出者

| 項目 | 内容 |
|---|---|
| 動機 | 恩義。PCに助けられた |
| PCとの関係 | 恩義→依存→対等。あるいは恩義→支配の固定化（PC態度次第） |
| 口調 | 最初はおどおど。PCキャラ次第で変化方向が異なる |
| シーンのトーン | 恩義からの提供か、対等な確認か、依存の表れか。この三つの区別が重要 |
| HP回復の描写根拠 | 精神的安らぎ。「誰かがいる」安心感 |
| 代償の描写 | 関係の曖昧さ。恋愛か恩義か共依存か不明確なまま身体だけが重なる |
| 離脱条件 | PCの行動に失望した時。自立して去る |

### 汎用NPC（娼館・集落）

テンプレート＋差分3項目で定義する。

| 項目 | 内容 |
|---|---|
| 口調 | 勢力圏に応じた差分（ニューエデン→商売的、ラケダイモーン→無感覚的） |
| 職業的距離感 | プロとしての態度と、その裏にある事情（借金、家族、選択肢の不在） |
| PC別反応 | ヴェスパ→同業者認識。アタルパ→信者への戸惑い。覇者→力への怯え |

## アナルの描写原則

膣との差異を必ず書き分ける。

| 要素 | 膣 | アナル |

| 背徳感 | 中 | 高。「本来の用途ではない」 |
| 嬌声 | 通常 | より大きく、より崩れやすい |

**PC別アナル特性：**

| PC（する側） | アナル時の特徴 |

| アタルパ | 「え、ここに入れるの？」。知識がない。相手の誘導で初めて知る |
