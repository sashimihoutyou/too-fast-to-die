# Archive

This directory stores completed design notes, superseded task lists, and old amendment documents.

Archive files are useful for provenance, but they are not source-of-truth documents. When an archive file conflicts with a current document in `docs/`, the current document wins.

## Layout

- `completed/`: designs or reviews whose implementation has already landed or whose actionable findings have moved elsewhere.
- `legacy/`: old task lists, reassessments, and amendment documents kept for history only.
