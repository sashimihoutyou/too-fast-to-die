## 16. 同行者システム

### 基本設計

- 同行者は基本的に一時的（期限付き、数ノード分）
- 期限内はパッシブ効果1つ＋専用カード1枚がデッキに追加
- 離脱時にカードも消失
- 期限が来たら別れ、カルマに応じてお礼（アイテム、情報、バイクパーツ）

### 同行者タイプと効果

| タイプ | リスク | パッシブ効果 | 専用カード |
|---|---|---|---|
| 戦闘型 | なし | 毎ターン敵全体に1ダメージ | 「援護射撃」（0AP・射撃・6ダメ・1戦闘1回） |
| 技術者 | 追手を抱えている（追加エンカウント発生） | バイク耐久の戦闘中消耗を無効化 | 「応急修理」（1AP・バイク耐久+3） |
| 商人 | 食料を大量消費（ノード移動ごとに燃料+1） | 次のショップで全品10%割引 | なし（マップ効果特化） |
| 情報屋 | なし | なし（戦闘効果なし） | なし。次の情報ノードで信頼度+1段階 |
| 足手まとい | 保護対象。死亡時カルマ大低下 | なし | 「かばう」（0AP・防御・被ダメを同行者HPに転送。同行者HP0で死亡） |
| 裏切り者 | 数ノード後にリソース窃盗＆逃亡。予兆あり（怪しい言動のテキスト） | 表向きは有用なパッシブ | 表向きは有用なカード |
| 犬 | なし | 夜間戦闘で先制1ターン獲得 | 「威嚇」（0AP・敵1体の次の攻撃を弱体化） |

犬は永続的同行者。放浪者の孤独パッシブを阻害しない。

裏切り者は必ず予兆テキスト（不審な言動）がイベント中に挿入され、注意深いプレイヤーは気付ける設計。完全なランダムペナルティにはしない。

### 同行者の加入条件

カルマ状態によって加入可能な同行者が制限される（→ 第21章参照）。

## 33. 固有同行者・固有NPCプロフィール

各PC専用の物語的に重要なキャラクター。「固有同行者」はバイクの後ろに乗る同行者枠を使用する。「固有NPC」は同行者枠を使用せず、イベントやエンディングで登場する。汎用同行者（娼婦／救出者／犬等）とは別枠。性的・身体プロフィールの詳細は `docs/pc-profiles.md` を参照。

**メインヒロイン制の廃止**：旧設計ではアタルパにノラ（専用娼婦同行者）、覇者にルト（専用常設同行者）を配していたが、ローグライト同行者システムとの構造的衝突により廃止した。固有の常設同行者はヴェスパ専用のマイロ（Act3以降・物語駆動）のみ残す。アタルパの信仰を揺るがす力は特定の女性ではなく旅そのもの（§32参照）。覇者の「生への引力」は道中の同行者ではなく目的地のNPC（ルト）が担う。

### 33-1.（廃止）旧ノラ → カルティスト固有同行者なし

アタルパには固有同行者を置かない。信仰を試すのは旅のすべて——風景、食事、安全、同行者の言葉、女の温もり、生き延びた実感——であり、特定の「誘惑者」キャラクターに集約しない。

汎用同行者（娼婦・救出者・村娘等）との性的関係は§29の枠組みで発生し、信仰度を削る。どの女と出会うかは周回ごとに異なり、ローグライトの再プレイ性と信仰揺らぎの多様性を両立する。

妊娠エンドはカルティスト固有のものとして残す。相手は固定キャラクターではなく、汎用同行者のうち性的関係が成立した女性。条件は世俗化中立（動揺〜懐疑）または棄教（離脱）ルートのみ。狂信（殉教＝死）ルートでは生じない。

信仰揺らぎイベント（§26）の「同行者の一言」は、その時点で同行している任意の同行者が発言する形で保持する。

### 33-2. ルト（覇者固有NPC・目的地）

旧設計では道中の常設同行者だったが、覇者の孤高（何も持たず誰も連れず歩く）との矛盾、および同行者枠の固定によるローグライト性の毀損を理由に、**オアシス到達後に出会う目的地NPC**に変更。

| 項目 | 内容 |
|---|---|
| 本名 | ルト（Lut）。覇者が「名を持たぬ男」であるのと対になる記号。最後に彼女が彼へ名を与える伏線 |
| 年齢・出身 | 21歳・女性。戦前の知識を守る小集落の出（壊滅）。**旅人の妹**＝覇者が運ぶ首飾りの持ち主の妹 |
| 経緯 | 読み書き・機械いじりを覚えた稀有な娘。だが「使える」ことが彼女を獲物にし、奪われ続けた。兄は病み、妹に死を見せまいと「オアシスを一目見たい」と東へ旅立ち、行き倒れた。その傍を通った覇者が首飾りを拾い東へ歩き始めた。残されたルトは兄を追い、先にオアシスにたどり着いていた |
| 人物 | 感情を言葉で説明せず行動で示す。「殴ること以外の全部ができる女」 |
| 求めるもの | 表層＝兄の消息。核＝奪われるのでなく与えること、そして「安全であるだけ」でなく「見られる」こと |
| 登場タイミング | オアシス到達後。覇者が首飾りを届けに来た時、受け取る側として現れる。兄の死をまだ知らない（あるいは薄々感づいている） |
| 旧同行効果の再配置 | 覇者の戦闘外の弱点（読み書き不可・バイク修理不可・知的交渉不可）はランダム同行者やイベントNPCが補う。道中の不自由さは覇者の孤高のコストとして残す |
| 性的関係 | あり。エンディング領域で発生。覇者の加害恐怖型に対し**ルトが誘う**（覇者は自分から動かない）。起点「**壊して…私を見て**」＝覇者の「壊す恐怖／detach／事後確認」を全反転 |
| 妊娠エンド | あり。**留まる・グッド側のみ**。死神（何も生まず死だけ撒く者）が生を創る＝テーマの完全反転。30-5の名づけエンディングと融合（名を持たぬ男の、名を持つ子） |

### 33-3. マイロ（ヴェスパ専用・唯一の固有常設同行者）

GDD 7-2の記載に加え、身長183cm（痩身長躯。ヴェスパ167cmより頭一つ高い）。詳細は `docs/pc-profiles.md`。

マイロはメインヒロイン制廃止後も残る**唯一の固有常設同行者**。理由：(1) Act3以降の合流であり、Act1-2はランダム同行者が完全に機能する (2) 合流自体がルート条件付きで、不在の周回も存在する (3) 禁断の性的関係ルートを含む複雑な関係性であり、単純な「ヒロイン枠」ではなく物語駆動の同行者である。

### 33-4. クロスオーバー：ミーシャ⇄ウェズリー

PCである両者は、互いのプレイ中に同行サブキャラとして登場する。

**ミーシャ（同行）＠放浪者ルート：**
- 犬が橋になる（獣を信じるミーシャは犬連れのウェズリーには警戒が緩む）＋父ジョエルの名で合流。犬あり/追求→合流成功（「ミーシャ＋虎＋犬」）、犬なし/不行動→「交差してまた離れる」悲劇（既定）。
- 同行効果：虎が戦闘に加わる。代償＝孤独パッシブの喪失（人間の同行者ゆえ）＝再接続の痛み。
- POV規律：ウェズリー視点ではミーシャを外側から描く＝彼の救済であり、彼女の「人間か獣か」の業はここで解決しない。
- エンディング：ジョエルの約束が「義務でなく生活」として果たされる。止まった腕時計が動く（場所でなく人が時間を動かす）。

**ウェズリー（同行）＠調教師ルート：**
- ミーシャは父の名に反応するが、ウェズリー本人とは面識がない（→ §7-3/7-4 タイムライン）。合流の糸は「父の名」と「犬」のみ。
- 同行効果：遠距離/銃の火力＋戦前知識・オアシス情報・文書の代読（ミーシャの銃器/重火器不可・知識の穴を補完）。
- エンディング：ミーシャ×ウェズリー妊娠エンド（30-7）はこの周回が本拠。

**性的関係（両ルート共通）：** ミーシャ18歳（修正指示書が統治）。年齢差（ウェズリー40代後半）と保護者的立場の非対称を地の文で常に意識させる（断罪はしない）。映える行為＝ウェズリーのクンニ・手技（前戯で壊す）／ミーシャ受け・困惑型絶頂。射精＝無言型。詳細は `docs/pc-profiles.md`。

### 33-5. 虎（ミーシャに内包）

放浪者ルートでは召喚ユニット。ミーシャ周回では常時パートナー（倒れても次戦闘で復帰）。名前をつけない（失う恐怖／所有の拒否／言語以前の信頼）。性的関係はミーシャ周回のみ（Misha×虎妊娠エンド・canon）。スキルの虎ルール（信頼の延長・擬人化しすぎない・単純な獣姦にしない）に従う。

## 34. サイドカー拡張スロット（新規設計）

### 設計背景

メインヒロイン問題（固定キャラが常に同行者枠を埋めてしまい、ローグライトの一期一会の同行者システムが機能しなくなる問題）の解決策として、バイクカスタマイズの新スロットを追加する。

### 仕様

| 項目 | 内容 |
|---|---|
| スロット名 | サイドカー |
| 分類 | バイクカスタマイズ枠の追加スロット（既存5枠＋1） |
| 効果 | 同行者をもう1人乗せられる（最大同行者数が1→2） |
| 入手 | 中盤以降（Act2-3）のショップ or イベント報酬 |
| デメリット | バイク移動速度低下（走行パートの逃走成功率-15%）、燃料消費+1/ノード |

### 設計意図

- ユニーク同行者と汎用同行者を同時に連れられるようになり、物語体験とゲームメカニクスが両立する
- サイドカー未入手の周回では従来通り1枠運用。入手はランダム性があるため、毎回同じ構成にならない
- デメリット（速度低下・燃費悪化）により、2人連れのリソース負荷が明確になる

## 35. ユニーク同行者プール（新規設計）

### 設計思想

PC固有の固定ヒロイン/ヒーローを廃止し、全PCで共有する「ユニーク同行者プール」を導入する。プール内のキャラクターは全PCのプレイで汎用同行者として出現するが、特定のPC×ユニーク同行者の組み合わせでのみ専用エンディングに到達する。

**例：マイロ**
- ヴェスパ以外のPCでプレイ中 → 飄々とした戦闘型の汎用ヒッチハイカーとして登場。専用セリフはあるが物語的には通常の同行者
- ヴェスパでプレイ中 → 同行中にマイロが弟であることが判明する物語展開が発生。専用エンディング分岐あり

この設計により：
1. **コンテンツ効率**: ベースの同行者データ（戦闘効果・パッシブ・汎用セリフ）は1回書くだけで全PC共通
2. **ローグライト性**: ユニーク同行者との遭遇はランダム。特定PCでプレイしていても会えない周回がある
3. **再プレイ動機**: 「あの同行者を別のPCで連れたらどうなるか」の好奇心が周回を促す
4. **専用エンディング**: PC×同行者の特定組み合わせのみ、追加の物語コンテンツとエンディング分岐が開放される

### ユニーク同行者一覧

| キャラ名 | 汎用タイプ | 汎用時の特徴 | 専用エンディングPC | 専用時の関係性 |
|---|---|---|---|---|
| マイロ (Mylo) | 戦闘型 | 飄々としたヒッチハイカー。鈍くさいが芯が強い | ヴェスパ | 生き別れた弟。姉弟の再会と禁忌 |
| ノラ (Nora) | 商人型 | 元娼婦の行商人。世慣れて逞しい | アタルパ | 信仰と世俗の境界を揺るがす女。殉教か生か |
| ルト (Lut) | 技術者型 | 読み書き・修理ができる若い女性 | 覇者 | ※道中同行者ではなく目的地NPC。§33-2参照 |

### 出現条件

- 各ユニーク同行者は Act2 以降のイベントノードでランダム出現（出現率: 1周回あたり平均2〜3人と遭遇）
- 専用エンディングPCでプレイ中のみ、出現率が若干上昇（+10%）
- 同じ周回で複数のユニーク同行者と遭遇可能（サイドカーがあれば同時に連れられる）

### 専用エンディング対応PC以外での扱い

ユニーク同行者は専用PC以外でも固有のセリフ・性格を持つが、物語的に深い関係には発展しない。汎用同行者としての期間が来れば離脱する（マイロ・ユキは期限なしだが、非専用PCでは5ノードで離脱）。
