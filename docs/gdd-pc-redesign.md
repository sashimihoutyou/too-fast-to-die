# プレイアブルキャラクター再設計

> 本ドキュメントは `gdd-characters.md` の §7（PC定義）、§19（獣カード）、§24（固有システムとコアループ）、§32（モチーフとテーマ）の **メカニクス部分** を置き換える。
> バックストーリー、動機、エンディング、信仰揺らぎイベント、追跡度システム、フレーバーテキスト等の **物語・イベント部分** は `gdd-characters.md` の記述が引き続き有効。
> ただし旧名（アタルパ→アータル、ヴェスパ→ワスプ）および覇者のPC削除は本ドキュメントが優先する。

---

## 全体方針

プレイアブルキャラクターを6人から5人に整理する。

| 旧 | 新 | 変更点 |
|---|---|---|
| カルティスト / アタルパ | **アータル** に改名 | 加速ゲージ → 速度/ギア制 |
| 元レイダー / ヴェスパ | **ワスプ** に改名（本名キイロは維持） | ヒート帯の再定義 + カード変質 + ランページ再設計 |
| 放浪者 / ウェズリー | 名前変更なし | 孤独パッシブ → 調査ゲージ + 情報開示 |
| 調教師 / ミーシャ | 名前変更なし | 獣召喚 → ポケモン式相棒スロット |
| 享楽者 / ホタル | 名前変更なし | エクスタシーにバズ帯追加 + 魅了ゲージ補給 |
| 覇者 | **PCから削除** | 拳法の達人系ユニーク同行者/NPCへ |

### 覇者のPC削除理由

- 闘気ゲージがワスプのヒートと構造が近い（溜めて転換点で強化）
- 銃・バイク不可がこのゲームのコアループ（バイク移動・燃料管理・銃カード）と噛み合わない
- 北斗の拳パロディ要素はPC枠を使わなくても成立する

覇者のNPC化については本ドキュメント末尾の §6 を参照。

## 1. アータル

### 名前変更

旧名アタルパを廃止し、アータルに改名する。語感を短くし、ワスプとの差別化を図る。火・燃焼・V8カルトの語感は残す。

### キャラコンセプト

V8カルトの若い巡礼者。真面目で誠実な少年期を終えたばかりの若者だが、世界の見方はカルトの教義に深く染まっている。

### 固有メカニクス: 速度 / ギア

旧加速ゲージ（0-30の連続値）を廃止し、1〜6の離散的なギア制にする。

| 項目 | 内容 |
|---|---|
| 速度 | 1〜6 |
| 戦闘開始時 | ギア1 |
| ターン開始時 | フルスロットル中でなければギア+1（自然加速） |
| ギア6到達 | フルスロットル発動 |
| フルスロットル終了時 | ギア1に戻る |

### ギア段階パッシブ

各ギアに常時パッシブ効果がある。ギアが上がるほど攻撃寄りになり、低ギアは安全だが弱い。

| ギア | パッシブ効果 |
|---|---|
| 1 | なし |
| 2 | バイクカードダメージ+1 |
| 3 | バイクカードダメージ+2、ターン終了時ブロック+2 |
| 4 | バイクカードダメージ+3、手札上限+1 |
| 5 | バイクカードダメージ+4、バイクカードAPコスト-1 |
| 6 | フルスロットル発動 |

ギア5はそれ自体がかなり強力であり、「ギア5で安定運転するか、ギア6に入ってバーストするか」の判断が生まれる。

### エンジンブレーキ（無料アクション）

| 項目 | 内容 |
|---|---|
| コスト | 0AP |
| 効果 | ギアを1段下げ、ブロック+3を得る |
| 制限 | 1ターンに1回 |
| 信仰離脱帯 | 使用不可（加速を制御する信仰が機能しない） |

守りたいターンにギアを落とす微調整手段。フルスロットルの到達を1ターン遅らせる代わりに即時防御を得る。

### アルティメット: フルスロットル

ギア6到達で自動発動する2ターン限定の加速状態。

| 項目 | 内容 |
|---|---|
| 発動条件 | ギア6到達（自動発動） |
| 継続 | 2ターン |
| 手札リロール | 即座に手札を全て捨て、手札上限+2で再ドロー |
| APコスト | 全カードAPコスト-1 |
| バイク燃料 | 発動ターンのみ全バイクカード燃料コスト0 |
| 終了時 | ギア1に戻る |

### 信仰度帯との連動

`gdd-characters.md` §26 の信仰度システムはそのまま有効。ギア制との接続は以下。

| 信仰度帯 | ギアへの影響 |
|---|---|
| 狂信（80-100） | 毎ターン自然加速+1に加え、バイクカード使用時にギア+1（最速2ターンでギア6） |
| 動揺（50-79） | 通常通り |
| 懐疑（20-49） | ギア上限5（フルスロットル発動不可。代わりにギア5パッシブの安定運用が戦略になる） |
| 離脱（0-19） | ギアシステム無効。エンジンブレーキも使用不可 |

### 専用カード方針

| 系統 | 内容 |
|---|---|
| 速度上昇カード | 弱いが即座にギア+1。早くフルスロットルへ入る |
| 速度低下カード | 強いがギア-1〜-2。今の戦況を有利にする。-2は消耗（Exhaust）付き |
| バイクカード | ギア段階に応じた追加ダメージ。燃料消費あり |

### バイクカード相性

アータル専用バイクカードはギアに応じてダメージが加算されるため、アータルは燃料を火力に変えるハイリスク・ハイリターン型になる。

### プレイ感

「ギアを上げて強力なパッシブを得るか、速度低下の強カードでギアを犠牲にするか」を毎ターン選ぶ。ギア5の維持 vs フルスロットルの爆発という二段階の目標がある。

### 外見

V8の聖印を背負った若い巡礼レーサー。

- 黒い作業着、または袖を切ったレーシングツナギ
- 儀式布や赤土で補修された巡礼装束
- 胸にV8の円形紋章。遠目には聖印、近くで見るとエンジン図面
- 首にスパークプラグや小型ピストンのロザリオ
- 右手に火傷痕。祈祷文を書いた耐熱布で巻いている
- 腰に小型燃料缶
- 左腕に壊れたタコメーター/ギア表示器
- 配色は黒、赤土、オイルの光沢、焼けたオレンジ

## 2. ワスプ

### 名前変更

旧名ヴェスパを廃止し、ワスプに改名する。ヴェスパより鋭く、レイダーのコードネームとして強い。本名キイロは維持する。

### キャラコンセプト

元レイダー。暴力で生き延び、殺すことに慣れすぎた若い女。ヒートは怒りではなく、暴力に戻っていく身体の癖。

### 固有メカニクス: ヒート

| 項目 | 内容 |
|---|---|
| 固有資源 | ヒート 0〜100 |
| 増加 | 銃カード、爆発カード、撃破、被弾 |
| 減少 | ブロックでダメージ軽減時（軽減量の30%、端数切捨）、冷静系カード、休息、特定イベント |
| 転換点 | 50以上で血の怒り、90以上で血の激昂 |
| アルティメット | ランページ（ヒート70以上で使用可能） |

### ヒート帯

| ヒート | 状態 | 効果 |
|---|---|---|
| 0-49 | 冷静 | 補正なし。防御/照準カードが安定 |
| 50-89 | 血の怒り | 攻撃ダメージ+2。手札の最もブロック値の高いカード1枚が「怒りの一撃」に変質（後述） |
| 90-100 | 血の激昂 | 攻撃ダメージ+5。ターン開始時HP-1。手札の全ブロックカードが「怒りの一撃」に変質。友好的選択肢制限 |

### カード変質: 怒りの一撃

ヒート50以上で、手札のブロックカードが攻撃カードに変質する。

| 変質条件 | 対象 |
|---|---|
| ヒート50-89 | 手札で最もブロック値の高いカード**1枚** |
| ヒート90-100 | 手札の**全ての**ブロックカード |

変質後の「怒りの一撃」:
- 元のブロック値がそのまま攻撃ダメージになる（ブロック8のカード → 8ダメージの攻撃カード）
- APコストは元のカードと同じ
- ブロック値は0になる
- ターン終了時に元に戻る（手札に残っていれば）

**設計意図:** ブロック-20%のような曖昧な数値ではなく、「カードが目の前で攻撃に変わる」という明確な代償。デッキ構築でブロック値の高いカードを入れると、高ヒート時に強力な攻撃に変わるシナジーも生まれる。

### 防御によるヒート冷却

ブロックでダメージを軽減した時、軽減量の30%（端数切捨て）のヒートを失う。

- 例: ブロック8で敵の10ダメージのうち8を軽減 → ヒート-2
- ヒート50以上ではカード変質で防御カードが減るため、「防御してヒートを下げたいのに防御カードがない」ジレンマが生まれる
- 冷静系カードをデッキに入れなくても、防御プレイで自然にヒートが下がる道がある

### アルティメット: ランページ

ヒート全消費ではなく、**60固定消費**。

| 項目 | 内容 |
|---|---|
| 使用条件 | ヒート70以上 |
| ヒート消費 | 60（固定） |
| ダメージ | 基礎25 + 残りヒート × 0.5（端数切捨） |
| 撃破時 | 残りヒートを10消費して別の敵にランページの半分のダメージで追撃 |
| 使用後 | 残りヒートはそのまま維持（冷静帯に戻ることもあれば、血の怒りに留まることもある） |

計算例:
- ヒート70で使用: 60消費→残り10→ダメージ25+5=30。残り10で冷静帯に復帰
- ヒート100で使用: 60消費→残り40→ダメージ25+20=45。残り40で冷静帯に復帰
- ヒート100で使用＋撃破: 残り40→追撃22→ヒート10消費→残り30で冷静帯

**設計意図:** 「ヒート70で早撃ちして安全に冷却するか、100まで粘って大ダメージを出すか」の判断。100まで粘ると血の激昂のカード変質で防御不能ターンが増えるため、実際のリスクがある。

### ヒート連動選択肢ロック

`gdd-characters.md` §7-2 の選択肢ロック仕様はそのまま有効（旧名ヴェスパ→ワスプに読み替え）。

### プレイ感

上げれば殺せる。上げすぎると防げない、止まれない、選べない。ヒートは火力資源であり、破滅の進行でもある。ブロックで冷却できるが、高ヒートではブロックカード自体が攻撃に変わる悪循環。

### 外見

黒黄の警告色をまとった元レイダーの刺客。

- 黒いレザージャケット
- 片肩だけスパイク。左右非対称
- 背中のコカトリス紋章を削り取った跡
- 黒いバンドゥ、またはコンバットブラ
- 黄色いラインは袖、ブーツ、ホルスター、バイクのタンクに少量
- 金髪ショート。片側刈り上げ、または黄色メッシュ
- 左目下の傷
- 太ももホルスターとナイフ
- スパイクは飾りではなく、近接戦用の武装として見える位置に置く

## 3. ウェズリー

### キャラコンセプト

孤独な放浪者から、私立探偵・証拠集めのキャラへ寄せる。戦前の記憶を持ち、カウボーイを演じながら、荒野で真実の断片を拾う男。

### 固有メカニクス: 調査ゲージ

旧「孤独パッシブ」を廃止し、調査ゲージに置き換える。

| 項目 | 内容 |
|---|---|
| 固有資源 | 調査ゲージ 5/5 |
| 減少 | ターン開始時-1 |
| 回復 | 調査回復カード、特定サーチカード |
| ゲージ0時 | 調査済みの敵1体の調査が全消去。ゲージ3で再開（5ではない） |
| アルティメット | Q.E.D. |

### 孤独パッシブとの統合

旧GDDの「同行者なしでサーチ強化」を調査ゲージに統合する。

| 同行者状態 | 調査ゲージ自然減少 |
|---|---|
| 同行者なし | 0（ゲージが減らない） |
| 犬のみ | 0（犬は調査を邪魔しない） |
| 同行者あり | -1/ターン（通常） |

**設計意図:** 同行者を取ると調査が時限式になり、取らないと無限に調査できる。「同行者のサポート vs 孤独の自由度」の根本的トレードオフ。

### 調査デバフ: 情報開示

敵に「調査」スタックを付与する。従来の単なる数値カウンターではなく、**スタックごとに敵の隠し情報が開示される**。

| 調査スタック | 開示される情報 |
|---|---|
| 1 | 次のターンの行動意図（インテント）が確定表示される（通常は現ターンのみ） |
| 2 | 弱点属性が表示される |
| 3 | ミッシング・リンク成立。全ステータス完全開示。Q.E.D.使用可能 |

**設計意図:** 探偵のファンタジー（情報を集めて暴く）がメカニクスで直接表現される。他キャラは敵の現ターンのインテントしか見えないが、ウェズリーだけが先読みできる。

### ミッシング・リンク

以下のどちらかで成立:
- 敵全体の調査スタック合計が5以上
- 単体の調査スタックが3以上

ミッシング・リンク成立中の敵に対してQ.E.D.が使用可能。

### アルティメット: Q.E.D.

| 項目 | 内容 |
|---|---|
| 使用条件 | ミッシング・リンク成立中の敵がいる |
| 対象 | ミッシング・リンク成立中の敵1体 |
| 効果 | 中ダメージ + 1ターンスタン + 2ターンガードブレイク |
| ガードブレイク | ブロック獲得不可、既存ブロック0固定 |

### 調査ゲージの消費先（二股構造）

調査ゲージはカード効果のコストとして消費される。消費先は二系統:

| 系統 | 効果 | ゲージ消費 |
|---|---|---|
| **カード系（デッキ操作）** | | |
| 捜索 | 山札からカード1枚を探して手札に加える | 1 |
| 再構成 | 捨札からカード1枚を回収 | 1 |
| **敵系（調査付与）** | | |
| 聞き込み | 敵1体に調査1付与 + 1ドロー | 1 |
| 推理 | 敵1体の調査スタックを別の敵に移す | 0 |

毎ターン「デッキを回すために使うか、敵の調査を進めるか」の判断が発生する。

### ゲージ0時のペナルティ

ゲージが0になった時:
1. 調査済みの敵のうちランダムに1体の調査スタックが**全消去**される
2. ゲージが**3で再開**する（最大値の5ではない）

**設計意図:** Q.E.D.使用前にゲージ切れすると致命的。ゲージ3で再開するため、長期戦ほどサイクルが短くなり不利（探偵は素早く解決すべき）。

### サーチカード系統

| 系統 | 効果 |
|---|---|
| 捜索 | 山札からカードを探す（調査ゲージ消費） |
| 証拠保全 | 調査ゲージ回復 |
| 聞き込み | 敵に調査付与 + 1ドロー（調査ゲージ消費） |
| 推理 | 調査スタックを別敵に移す |
| 再構成 | 捨札からカードを回収（調査ゲージ消費） |

### プレイ感

ゲージは増やすものではなく、減っていくもの。消える前に証拠を繋げるか、サーチに使ってデッキを回すかを選ぶ。同行者なしならゲージは無限だが体力がきつい。硬い敵に強く、再現性が高い。

### 外見

カウボーイを演じる戦前生まれの私立探偵。

- ダスターコート
- カウボーイハット
- コート内側に手帳、証拠袋、折れた鉛筆、古い写真
- 古いフィルムカメラ、または壊れかけのポラロイド
- ベルトに小型ルーペ
- 低い位置のリボルバーホルスター
- 帽子のつばに小さな弾痕
- 配色は褪せた紺、砂色、革の茶
- 犬同行時は足元に犬。犬がいない時は影が広く見える構図

## 4. ミーシャ

### キャラコンセプト

調教師ではなく、群れの中心にいる子。獣を命令するのではなく、動物カードで相棒をセットし、指示カードで操作する「ポケモン式」のキャラ。

### 固有メカニクス: 相棒スロット

#### 基本構造

| 項目 | 内容 |
|---|---|
| 相棒スロット | 初期1枠。レリックで最大3枠まで拡張可能 |
| 動物カード | デッキに統合。使用すると相棒スロットにセットされる |
| 自動攻撃 | 毎ターン開始時、場の相棒が自動で敵1体を攻撃 |
| ダメージ肩代わり | ミーシャへのダメージを相棒と均等割り |
| 手動回収 | スロットクリックで相棒を手札に戻せる（使用APは返らない） |

#### ダメージ肩代わり（均等割り）

ミーシャへのダメージは、ミーシャと場の全相棒で均等に分配する。

| 相棒数 | ミーシャ負担 | 各相棒負担 |
|---|---|---|
| 0体 | 100% | — |
| 1体 | 50% | 50% |
| 2体 | 34% | 33%ずつ |
| 3体 | 25% | 25%ずつ |

各相棒はガード値（常時ダメージ軽減）を持ち、肩代わりしたダメージからガード値を引いた分がHPに入る。

#### 動物カード

| 動物 | AP | HP | ダメージ/ターン | ガード | 特性 |
|---|---|---|---|---|---|
| 虎 | 3 | 20 | 5 | 2 | **戦闘開始時セット済み**。HP0で退避、次戦闘でHP半分で復帰 |
| デビルフ | 2 | 8 | 8 | 0 | 速攻型。セットしたターンも攻撃する |
| 変異グリズリー | 3 | 30 | 4 | 5 | 重装型。ガードが高い |
| 汚染イーグル | 2 | 6 | 3 | 0 | 飛行。AoE攻撃の肩代わり対象にならない。ランダム対象攻撃 |
| アーマージロ | 2 | 25 | 0 | 8 | 攻撃しない。肩代わり率が通常の均等割りではなく75%固定 |

**ガードの定義:** ブロックではなく常時ダメージ軽減。虎のガード2は、肩代わりしたダメージから毎回2を引く。小ダメージの連打に強く、大ダメージの一撃に弱い。

#### 虎の特別ルール

- 戦闘開始時に自動で相棒スロットにセット（召喚コストなし）
- HP0で「退避」（その戦闘では使用不可）。カードは捨札に送られる
- 次の戦闘でHP半分で復帰（再び自動セット）
- 虎のみ、HP0になっても永久消滅しない

#### 相棒HP0時の処理

虎以外の動物がHP0になった場合:
- カードが捨札に送られる
- 再度手札に引けば、満タンHPで再セット可能
- 永久消滅はしない（消耗扱いではない）

#### 相棒スロット拡張

レリックで拡張。ランごとに「今回は2体ラン」「今回は3体ラン」が変わるローグライト的なバリエーション。

| レリック | 効果 |
|---|---|
| 壊れた首輪 | 相棒枠+1（最大2体） |
| 群れの証 | 相棒枠+1（壊れた首輪とは別枠。最大3体） |

#### 指示カード

指示は全相棒に適用される。

| カード | AP | 効果 |
|---|---|---|
| 攻撃指示 | 1 | このターン全相棒のダメージ2倍、ガード0 |
| 防御指示 | 1 | このターン全相棒のガード2倍、全被弾を相棒が均等負担（ミーシャ被弾0） |

複数相棒時に指示カードの価値がスケールする。3体に攻撃指示＝3体分の2倍ダメージだが、ガード0の3体に被弾が来たら壊滅する。

### アルティメット: 群れの咆哮

| 項目 | 内容 |
|---|---|
| 発動条件 | 相棒がセットされてから**3ターン生存** + デッキまたは手札に動物カード1枚以上 |
| AP | 0 |
| 再発動 | アルティメット使用後や相棒が倒された後は、再セットから3ターン生存で再び使用可能 |

#### 効果（1ターン限定の大爆発）

1. **サーチ:** デッキと捨札から全ての動物カードを手札に加える
2. **このターンのみ:** 手札の各動物カードのステータスが反映される
   - 手札動物の**攻撃力** → 場の相棒の攻撃に**加算**（全動物が一斉に襲いかかるイメージ）
   - 手札動物の**ガード** → ミーシャの**ブロックに加算**（群れがミーシャを囲んで守るイメージ）
3. 場の相棒が**即座に1回攻撃**する（加算済みの攻撃力で）
4. **次のターン開始時:** 相棒スロットの動物と手札の動物カードが**全て捨札**に送られる（動物たちが疲弊し、隙ができる）

#### 具体例

**相棒枠1、虎が場、デビルフ・グリズリー・アーマージロがデッキにいる場合:**

- 3枚サーチして手札へ
- 手札攻撃力合計: デビルフ8 + グリズリー4 + アーマージロ0 = **+12**
- 手札ガード合計: デビルフ0 + グリズリー5 + アーマージロ8 = **ブロック+13**
- 虎が即時攻撃: 5（素）+ 12（群れ）= **17ダメージ**
- ミーシャにブロック**13**付与
- 次ターン開始時: 虎含む全動物が捨札へ。ミーシャは丸裸

### 専用カード: 指笛

アルティメットのデメリット（全動物捨札）からの回復手段。

| 項目 | 内容 |
|---|---|
| 名前 | 指笛 |
| AP | 1（強化後: 0） |
| 効果 | 捨札の動物カード1枚をランダムに手札に加える。捨札に動物カードがなければデッキから1枚ランダムにサーチ |
| 初期デッキ | 2枚 |

**設計意図:** アルティメット後の「群れの咆哮→全動物捨札→指笛で回収→虎再セット」復帰ルートを確保。通常戦闘中にも「虎が倒された→指笛で回収→再セット」として機能する。ランダムなので常に最適解は出ない。

### 獣カードの入手方法

- 屈服の鞭で獣系の敵にトドメ → 確定入手
- 獣系の敵を通常撃破 → 30%で入手
- 特定イベント → 選択肢で入手

| 虎 | 1 | 3 | 相棒セット。5ダメ/ターン、ガード2。戦闘開始時セット済み |

### プレイ感

動物をセットして指示を出す。守れば長く使える。攻撃指示で押せば高火力だが相棒が傷つく。群れの咆哮で全動物を一斉動員できるが、翌ターンは丸裸。指笛で立て直す。

### 外見

虎と同じ檻から出た、小さな群れの中心。

- 布、革、虎の抜け毛、金属片を縫い合わせた短い外套
- 肩から虎の古い毛皮片を掛ける
- 首と手首に拘束具の痕
- 片手首に外れた鎖の輪を残す
- 腰に合図用の骨笛、革紐、獣の爪、餌袋
- 足元は布を巻いた足、または硬くなった裸足
- 黒く長い髪を骨や羽根で束ねる
- 目はアンバー
- 虎にも同じ首輪痕がある

## 5. ホタル

### キャラコンセプト

依存でしか感情が起動しない女。快楽を維持すれば強いが、100に届くと暴発する。

### 固有メカニクス: エクスタシー

| 項目 | 内容 |
|---|---|
| 固有資源 | エクスタシー 0〜100 |
| 増加 | ネオン系カード、魅了系カード、同行者、魅了状態の敵 |
| 減少 | 毎ターン自然減衰-8（戦闘中）、クラッシュ系カード |
| 転換点 | 60以上でバズ、75以上でNeon High、100でクライマックス |
| アルティメット | クライマックス（ゲージ100で強制発動） |

### ゲージ帯

| ゲージ | 状態 | 効果 |
|---|---|---|
| 0-9 | 重度離脱 | AP-1、毎ターンHP-1、手札-1 |
| 10-32 | 離脱 | AP-1 |
| 33-59 | 通常 | 補正なし |
| 60-74 | バズ | ドロー+1。ただしターン終了時に手札を1枚ランダムで消耗（Exhaust） |
| 75-94 | Neon High | AP+1、ドロー+1 |
| 95-99 | 危険域 | Neon High継続。ネオン系カード使用でクライマックス確定発動 |
| 100 | クライマックス | 強制発動 |

**バズ帯（60-74）の設計意図:** 通常帯とNeon Highの間のグラデーション。ドロー+1でカードを多く見れるが、ランダム消耗でデッキが痩せていく。「バズ帯で止めてデッキを回す」プレイスタイルか、「Neon Highまで上げて安定させる」かの選択が生まれる。

### 戦闘中の自然減衰

**毎ターン開始時にエクスタシー-8。**

- Neon High（75）からは約5ターンで通常帯に落ちる
- ネオン系カード1枚（+10〜+15）で1ターン分の減衰を相殺
- 75を維持するには2〜3ターンに1回はネオン系カードを使う必要がある

### 魅了状態の敵によるゲージ補給

魅了状態の敵が場にいる間、**ターン開始時にエクスタシー+3/魅了された敵1体**。

- 魅了をゲージ維持手段として機能させる
- 「敵を魅了して場に残す」vs「敵を倒して戦闘を終わらせる」の判断が生まれる
- ホタルのテーマ（依存先を確保し続ける）とメカニクスが合致

### アルティメット: クライマックス

ゲージ100到達で強制発動する1ターン限定の爆発。

| 項目 | 内容 |
|---|---|
| 発動条件 | ゲージ100到達（自動発動）。95-99で ネオン系カード使用でも発動 |
| 発動時 | 手札を全て捨て、7枚ドロー。このターン全カードAP0、全ダメージ2倍 |
| ターン終了時 | ゲージ0にリセット |
| 殲滅成功 | オーバードーズなし。次の戦闘開始時ゲージ40 |
| 殲滅失敗 | 次ターンにオーバードーズ発動。次の戦闘開始時ゲージ10 |

### オーバードーズ

クライマックスで全敵を殲滅できなかった場合、次ターン開始時に発動。

| 項目 | 内容 |
|---|---|
| AP | 0（行動不能） |
| HP | 現在HPを半分にする（端数切上） |
| 最大HP | 永続的に-3（ラン中蓄積） |
| ゲージ | 0のまま → 重度離脱に突入 |

### アフターグロウ（戦闘間の残響）

| 状況 | 次の戦闘の開始時ゲージ |
|---|---|
| クライマックスで殲滅成功 | 40（通常帯で安定スタート） |
| クライマックスで殲滅失敗（オーバードーズ経由） | 10（離脱帯からスタート） |
| クライマックスなしで戦闘終了 | 前回の戦闘終了時の値を維持 |

**設計意図:** クライマックス→0→重度離脱→次戦闘も重度離脱のデススパイラルを緩和。殲滅成功なら40スタートで通常帯、失敗でも10で離脱帯（重度離脱ではない）。

### 魅了

ホタル固有の状態異常。敵にスタックを蓄積する。

| スタック | 効果 |
|---|---|
| 1 | 敵の攻撃力-3 |
| 2 | 敵の攻撃力-5。敵の行動順が最後になる |
| 3 | 愛の奴隷の対象条件。魅了は自然減少しない |

### 愛の奴隷

| 項目 | 内容 |
|---|---|
| 使用条件 | 魅了3以上の人型（非ロボット・非ボス）敵がいる時、自動で手札に追加 |
| 効果 | 対象を戦闘から除外し、同行者として獲得 |
| 制限 | 1戦闘につき1回 |

### マップ上でのゲージ変動

| 状況 | ゲージ変動 |
|---|---|
| 同行者あり（ノード移動ごと） | +6 |
| 休息ノード | +15 |
| 享楽イベント（歓楽街等） | +20〜30 |
| ネオン入手イベント | +10 |
| 同行者離脱時 | -1 |
| 自然減衰（戦闘なし区間） | -3/ノード |

### プレイ感

60-94を維持すると強い。100に届くと爆発する。爆発で決めきれなければ壊れる。魅了した敵をゲージ補給源として利用しつつ、Neon Highを維持するゲージ管理が核。

### 外見

ネオンで光る、壊れた歓楽街のバニー。

- 年齢不詳だが、外見は成人女性寄り
- バニースーツは歓楽街の衣装を防刃素材とベルトで改造したもの
- ウサ耳は片方折れている
- 黒いレオタードに蛍光ピンク/黄の補修糸
- 首にネオンバイアルのペンダント
- 目元、鎖骨、指先に薄い蛍光
- 黒髪ベースに蛍光ピンクと黄色のエクステ
- 根元が伸びていて、旅で手入れできていない
- 厚底ブーツの踵が削れて傾いている
- 腰にネオンケース、化粧ポーチ、小型拳銃
- 笑顔が明るいほど怖い

## 6. 覇者（NPC化）

### 方針

PCから削除し、荒野で出会う拳法の達人系ユニーク同行者として再利用する。

### 再利用案

| 項目 | 内容 |
|---|---|
| 呼称 | 覇者 / 名なしの男 |
| 加入条件 | 荒野での一騎打ち、奴隷闘技場、格闘イベント |
| 同行効果 | 戦闘開始時、敵1体に威圧または低ダメージ |
| 離脱条件 | 離脱ポイント5蓄積で無言で去る |

### 百裂拳カード

| 項目 | 内容 |
|---|---|
| コスト | 3AP |
| 効果 | 単体に15ダメージ×3ヒット。消耗 |
| ボス | ヒット数が2に減少 |
| 追加 | 使用後、覇者が1ターン「構え」に入り、敵1体の次の攻撃を無効化 |

### 離脱ポイント

| 行動 | 離脱ポイント |
|---|---|
| 弱者を見捨てた | +3 |
| 覇者の前で奴隷を買った | +2 |
| 暴力的選択肢（残虐行為） | +1 |
| 5ポイント蓄積 | 覇者が無言で去る |

### 物語的役割

覇者の魅力は「無言で殴る」「常識が通じない」「荒野に突然現れる異物感」にある。PCではなく一点物のNPCとして扱う方が、この異物感が強く残る。バックストーリー、ルトとの関係、旅人の首飾り等の設定は `gdd-characters.md` §7-5, §27-6, §27-7 に記載のまま有効（ただしPC固有システム・闘気ゲージ・闘気呼法の記述は無効）。
