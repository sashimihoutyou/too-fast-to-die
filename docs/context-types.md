## コンテキスト別 詳細テンプレート

### 享楽（自発的）

**発生場面：** ニューエデン歓楽街、酒場2F、集落の宴、V8カルトの共同体内

**標準3幕を使用。**

第1幕の重点：
- なぜPCがここにいるのか。逃避か、報酬か、好奇心か
- 環境の活力。音楽・酒・人混み・熱気
- 相手との接触のきっかけ

第2幕の重点：
- 高濃度。身体描写を丁寧に
- PCのキャラクター性が最も自由に出る場面
- 環境音を混ぜ続ける（壁越しの笑い声、遠くの音楽）
- クライマックスを明確に描く。射精/絶頂/限界到達を、台詞・呼吸・身体反応・相手の反応で処理する
- 胸、ペニス、体格差、傷跡、経験値など、キャラ固有の身体特徴を行為の進行に反映する

第3幕の重点：
- 世界への帰還。酒場を出ると荒野の風が吹く
- ゲーム的結果との接続（HP回復、信仰度変動等）
- 享楽の後の虚しさ、あるいは活力の回復
- 事後描写または第二ラウンドへの移行を書く。高濃度シーンではクライマックス直後に暗転して終わらせない

**文字数目安：** 1500〜3000字

### 取引的

**発生場面：** 借金娼館、燃料/ネオンとの交換、サンドバイパーズ通行料代替、情報取引

**簡略2幕を使用。**

第1幕の重点：
- 取引の構造を明示する。何と何が交換されるのか
- PCの内面：割り切りか、罪悪感か、慣れか
- 相手の事情を示唆する（借金、飢え、選択肢の不在）

第2幕の重点：
- 行為は圧縮。心理が主体
- 事後の描写に力を入れる。「金を払った」あとの空気
- カルマへの影響を意識

**文字数目安：** 500〜1500字

---

### 同行者との親密

**発生場面：** 宿泊イベント、休息ノード

**標準3幕を使用。関係深度で濃度が変動する。**

| 関係段階 | 濃度 | トーン |
|---|---|---|
| 初回 | 中 | 距離感。探り合い |
| 中盤 | 中〜高 | 馴染み。個人的なやり取りが入る |
| 深化後 | 高 | 関係の質が変わる。名前の呼び方が変わる |

**娼婦同行者の場合：** 取引→馴染みへの移行が軸。プロとしての距離が縮まる過程。

**救出者同行者の場合：** 恩義→対等への移行が軸。対等にならない場合、依存の描写が入る。

**高濃度の同行者シーン：**
初回以外は、クライマックス後に事後の会話か第二ラウンドを置く。関係深化後の第二ラウンドは、一回目より遠慮が薄れ、呼び方・触れ方・声の出方が変化する。

### 制度的

**発生場面：** ラケダイモーン繁殖施設、ダストランナー婚姻仲介

**簡略2幕 or 暗転型。**

- 個人の感情が制度に押し潰される空気を優先
- 効率的で感情を排除した性。「作業」としての身体接触
- PCの内面：非人間性の目撃と、自分がその一部になる恐怖
- ダストランナー婚姻仲介の場合：善意と売買の境界の曖昧さ

**文字数目安：** 300〜1500字

---

### 暴力隣接

**発生場面：** コカトリス捕獲、奴隷制下の強制、レイダーの行為

**構造は場面次第。濃度制限なし。**

PC被害の場合：
- 脱出/逆転の余地を維持する。完全な絶望で終わらせない
- 身体の描写と心理の描写を交互に。身体が先に反応し、心理が追いつく/追いつかない
- 事後の身体：痕跡の描写。傷・痛み・汚れ。「何があったか」の物証
- クライマックスを描く場合は、快感ではなく身体の裏切りや恐怖との乖離として扱う。描写の焦点はPCの認識と脱出/逆転の余地に置く

PC加害の場合：
- カルマ大幅減。ゲーム的結果を意識する
- PCの内面を描くか描かないかが重要な選択。描かない（無感覚）は低カルマPCの表現
- 被害NPCの反応を個体化する。恐怖・怒り・諦め・無感覚
- 事後描写を省略しない。PCが痕跡を見るか見ないか、相手の状態を確認するか無視するかでカルマ帯を表現する

### 変異種・動物

**発生場面：** ブルードマザー捕獲、デビルフ群れ、調教師固有イベント

**標準3幕 or 簡略2幕。**

ブルードマザー（PC捕獲）：
| 幕 | 内容 |
|---|---|
| 第1幕 | 捕獲の過程。巣への搬入。環境描写（粘液・暗闇・体温・湿度） |
| 第2幕 | 繁殖行為。本能のみの接触。PCの恐怖と身体の反応の乖離 |
| 第3幕 | 脱出。事後の身体。粘液を拭う。世界に戻る |

デビルフ（群れ）：
- 群れの階層構造が行為に反映（アルファが先）
- 獣の本能。コミュニケーション不在
- 調教師の特例：群れとの関係構築ルートで根本的にトーンが変わる

調教師固有（虎）：
- 信頼の延長。恐怖がない唯一の変異獣シーン
- 擬人化しすぎない。獣としての反応と、ミーシャの人間としての感情の非対称性
- 環境：荒野の夜。焚火の残り火。二人きりの安全圏
- 高濃度化する場合も、単純な獣姦として処理しない。体温、心拍、毛皮、重さ、呼吸、ミーシャの安心感を軸に、クライマックス後は虎のそばに戻る事後描写まで置く

**文字数目安：** 1000〜2500字

### 目撃シーン

**発生場面：** 奴隷暴行、レイダーの行為、娼館の裏側、繁殖施設、市場

**目撃シーン固有の構造：**

| 要素 | 描写方法 |
|---|---|
| PCの視線 | 何を見ているか。どこに目が行くか。目を逸らすか逸らさないか |
| PCの身体反応 | 呼吸・手の位置・筋肉の緊張。武器に手が伸びるか |
| 行為の描写 | PCの視界に入る範囲で。全容を描写しない。断片的な情報 |
| 音 | 行為の音。PCの呼吸音。環境音。沈黙 |

**介入可の場合：**
- 介入の瞬間を丁寧に書く。迷い→決断→行動の過程
- 介入しない場合も同じ密度で「しない」決断を書く
- 介入結果は必ずしも良くない。助けた相手が感謝しないケースを含める

**介入不可の場合：**
- 無力さの描写。だが「何もできない」で終わらせない
- この光景がPCの以後の選択に影響する種を蒔く

**PC別目撃反応テンプレート：** → `pc-profiles.md` 参照

**文字数目安：** 300〜1000字

### ネオン使用中

**通常のコンテキストに「知覚変容レイヤー」を追加する。**

| ネオン使用量 | 追加描写 |
|---|---|
| 軽度 | 五感描写が鮮烈に。通常なら気づかない微細な感覚を拾う。色が鮮やかに、音が近くに |
| 重度 | 感覚の乖離。自分の身体を外から見ている感覚。行為は認識できるが触感が「膜越し」 |
| 過剰 | 段落間の断裂。時間が飛ぶ。PCが目覚めて自分の身体の状態から逆算する |

**文字数：** 通常の1.2倍（変容描写の加算分）

**ホタル固有：** ネオン使用中の性的シーンでは、ホタルの知覚変容が他PCより顕著に描写される。通常1.2倍の文字数上乗せに加え、感覚の混線（色が聞こえる、音が見える等のシネスタジア的描写）を入れる。ネオンが切れた状態との落差を対比として意識する。

## PC別フレーバー差分

各コンテキストでPC固有のフレーバーが挿入される場合の指針。

### ワスプ：ヒート連動

ヒート50%以上の享楽シーン：攻撃的な描写が混入する。「手加減しない」固有分岐。
ヒート90%以上の享楽シーン：暴力と性が混在する。カルマ-2〜-3（暴力×性の描写原則に準拠）。

### ホタル：ネオン依存と異様さ

**ネオンが切れた時の落差（断章用）：** 「先ほどまで踊りながらバイクに乗っていたホタルが、道端に蹲っている。嘔吐。震え。涙。蛍光ピンクのエクステが吐瀉物にかかっている。5分後、ネオンを見つけて吸い込んだ瞬間、立ち上がって笑った」

**クライマックス発動時：** ホタルの瞳孔が完全に散大する。快楽と殺意の区別がつかない表情で、ジュリエットを抜く。

**オーバードーズ（クライマックス翌ターン）：** ホタルは床に座り込んでいる。手が震えている。瞳孔はまだ散大している。

### アータル：信仰度帯差分

信仰度帯に応じてすべてのコンテキストの地の文トーンが変化する。

| 信仰度帯 | 性的シーンの特徴 |
|---|---|
| 狂信 | 行為を「加速の儀式」として正当化。事後にV8ペンダントに口づけ |
| 動揺 | 快楽を感じてしまう自分と教義の間で揺れる。「……罪だ」 |
| 懐疑 | 信仰の殻が剥がれ、生身の人間として感じ始める |
| 離脱 | 空洞。何をしても何も来ない |
