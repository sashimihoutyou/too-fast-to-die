## LLM的表現の禁止

LLMが大量生成する、中身のない型に誘惑されない。書き上げたら必ずこの節で点検する。

次の言い回しは、論点を増やさず「ちゃんと書いている感」だけを付けるLLM口調である。使わない。

| 分類 | 禁止表現 |
|---|---|
| 予告と総括 | 「重要なのは〜である」「ここでは〜について見ていく」「まとめると」「要するに」（直前の言い換えだけのとき）、「〜に他ならない」 |
| 正面から系 | 「正面から扱う/回収する/見る/書く/立てる」——中身の代わりに姿勢だけを宣言する |
| 空虚な形容 | 「不可欠」「核心的」「鍵となる」「根本的な」（主張の中身を説明せず強調だけする）、「多角的」「包括的」「総合的」（何をどう見たかを書かない） |
| 空虚な動詞 | 「掘り下げる」「深掘りする」「言語化する」（何をどう書いたかを示さず終わる）、「触れる」「言及する」（一段落で済ませるだけ） |
| 接続の型 | 「〜において」「〜という側面から」「〜の観点から」（新情報なし）、「さらに」「また」「加えて」の連打 |
| 弱い緩和と称賛 | 根拠なき「〜と言えるだろう」「〜かもしれない」、「非常に」「極めて」「大いに」（中身のない強調） |

**地の文での適用：** シーン・イベントの地の文にこれらが混入すると「AIが書いた文章」感が一気に出る。主張はそのまま書く。

**台詞での例外：** キャラの口癖や性格として「まとめると」等を使う場合は許容する。ただしそれがキャラ固有の語彙であることが明確な場合に限る。

## 推量と断定の区別

- 「かもしれない」「だろう」「ようだ」「らしい」は、根拠なく主張を弱めている場合だけ削る
- 作中人物の認識、状況からの推定、読者が抱きそうな疑念を表す場合は、不確実性を保つ
- 断定に直せるのは、テキスト内の根拠で命題が確定している場合に限る

**三人称ナレーションへの適用：** POVキャラが確信していないことを地の文で断定しない。「〜のように見えた」「〜だと感じた」と「〜だった」を使い分ける。相手の内面は行動・表情・声から推測させ、地の文で断定しない。

```
# NG: 相手の内面を地の文で断定
ウェズリーは彼女を哀れんでいた。

# OK: 外面から推測させる
ウェズリーの目がわずかに細まった。哀れみか、それとも別の何かか。
```

## 術語の一貫性

世界観固有の用語を導入したら、以後はその語で通す。曖昧語に後退しない。

| 導入した術語 | 後退してはいけない曖昧語 |
|---|---|
| ネオン | 薬物、ドラッグ、クスリ |
| カルマ | 評判、善悪値、道徳 |
| ヒート | 怒り、興奮度 |
| スピード・デーモン | 神、神様（V8カルトの文脈） |
| オアシス | 楽園、約束の地（目的地としての固有名） |
| AP | 行動力、アクションポイント（カード説明の文脈） |
| ブロック | 防御、ガード（戦闘の文脈） |

台詞中では、そのキャラが術語を知らない場合に限り一般語を使ってよい（例：V8カルト外のキャラが「スピード・デーモン」ではなく「あんたらの神さん」と言う）。

## 冗長の排除

- 同じ主張を言い換えて繰り返さない。一つの情報は一度だけ書く
- 場面を描写した直後にその内容を要約し直さない。意味づけの一文だけを置く
- 想像上の読者との問答（問いを立てて一語で答える形）を修辞として使わない。主張はそのまま述べる
- メタな枠取り（「ここまでの話には自然な続きがある」「〜という発想である」）で紹介しない。その内容を直接書く
- 読者が自力で補える中間段階の説明は書かない
- 接続や評価のためだけの文（「それ自体はよいことである」など）を置かない

**シーンへの適用：**
```
# NG: 描写の直後に要約
砂嵐が収まり、空に星が戻った。ウェズリーは焚火に薪をくべた。
つまり、嵐が去って夜の静寂が戻ったのだ。

# OK: 描写だけで十分
砂嵐が収まり、空に星が戻った。ウェズリーは焚火に薪をくべた。
```

## 演出の節度

演出の規範は全面禁止ではなく、節度の規範である。修辞は、それが効果を生む箇所でのみ使う。

- 溜め（「ここには〜が潜んでいる」）や修辞疑問で導出を演出するのは、緊張が議論に効く要所に限る
- 短い決め台詞を独立した段落にして緊張を作る演出を多用しない。体言止めは場面の山場に限り使ってよい
- 慣用表現をひねった言い回し（「知識を体に入れる」など）や、指す内容が一意に決まらない比喩を使わない。平易な動詞でそのまま言う
- 帰結の列挙によって事故や危険を煽らない
- 「AではなくBだった」という対句の決め台詞を多用しない

---

## メタ説明の完全排除

システム用語・読者への解説・状態報告・先読み予告を地の文に書かない。

```
# NG: メタ説明
アタルパの信仰度が高いため、ここでは祈りの台詞が入る。
この選択はカルマに影響する。

# OK: 物語として書く
アタルパは膝をつき、両手を組んだ。「スピード・デーモンよ、この試練を……」
```

## カード説明文

カード説明文は**機能の正確な記述**が最優先。フレーバーより明快さを取る。

- 効果を簡潔に。数値は具体的に書く
- 未実装の効果を記載しない（CLAUDE.mdの「説明文が嘘になるバグ」防止）
- 複数効果は読点で区切る。接続詞は省く
- 条件付き効果は条件を先に書く
- 強化値は実装済みの場合のみ記載する

```
# 良い例
"8ダメ、闘気+10"
"4ダメ（ブロック無視）。獣系にとどめで獣カード確定入手"
"ヒート÷3のダメージ（100%時≒33）"
"3T: AP0以下でもカード使用可。AP0以下で与ダメ+3。AP負債1毎にHP-3"

# 悪い例
"敵に強力な一撃を放つ（8ダメージ）"     ← 修飾が冗長
"闘気を溜めて戦う覇者の基本技"           ← フレーバーが効果を曖昧にする
"萎縮2付与"                              ← status_effectが未実装なら嘘
```

## イベント・選択肢テキスト

### イベント本文

- 世界観に接地する。環境描写（砂、熱、匂い、音）を必ず含める
- 状況を提示し、PCの選択に必要な情報を過不足なく与える
- 結果を暗示してよいが、明示しない（「この男は信用できそうにない」はOK、「信用するとカルマが下がる」はNG）

### 選択肢

- 選択肢の文面はPCの行動を示す。結果を示さない
- 口調はPCの性格に合わせる（覇者なら直截、ウェズリーなら抑制的）
- 選択不能な選択肢（条件未達）はグレーアウト表示を前提とし、テキスト上で理由を説明しない

```
# 良い例
「燃料を渡して通してもらう」
「力ずくで突破する」
「引き返す」

# 悪い例
「燃料を渡す（燃料-10）」       ← コストをテキストに含めない。UIが表示する
「友好的に話しかける（成功率60%）」  ← 確率をテキストに含めない
```

## 会話・台詞

### 口調の分離

PC間の口調を混同しない。各PCの口調は `docs/pc-profiles.md` に従う。

- アタルパ：素朴、信仰が滲む。敬語と砕けた言葉が混在
- ヴェスパ：粗野、攻撃的。虚勢。マイロの前でだけ動揺する
- ウェズリー：静か、簡潔。30年の重みが言葉の少なさに出る
- ミーシャ：言葉が足りない。人間の会話に不慣れ。獣の話になると饒舌
- 覇者：端的。力の論理。無駄な言葉を嫌う
- ホタル：完全にギャル。「ウケる」「マジ無理」「えっちすぎん？」。距離感ゼロ。下ネタに一切の躊躇がない。ネオンが切れると断片的な呻きと泣き声に変わる。この二つの声を混同しない

### 会話の原則

- 一人の台詞が長くなりすぎない。長い説明が必要なら、相手のリアクションで区切る
- 「〜だ」「〜である」調の講義口調を避ける。会話は会話として書く
- 情報伝達だけの会話にしない。やりとりの中にキャラ間の力関係・感情を滲ませる

```
# NG: 講義口調
「ネオンは旧世界の合成麻薬だ。神経系に作用し、感覚を鋭敏化させる。
しかし過剰摂取すると記憶障害を引き起こす。使用には注意が必要だ」

# OK: キャラの声で
「ネオン？　あれは脳みそを焼くぜ。一発目は天国、二発目は地獄だ」
「……使ったことがあるの？」
「馬鹿言え。使った奴の末路を見てるだけで十分だ」
```

# 第3部：シーン執筆

シーン（戦闘外の物語パート）全般に適用する。性的シーン固有のルールは第4部。

---

## シーン文体原則

- **ト書き禁止。** シーンは小説として書く
  - 行為の過程を感覚・心理・環境と編み合わせて時間軸で描写
- **行為と反応を一文に編む**
  - 「触れた。震えた。」ではなく「指先が肌に触れた瞬間、背中が波打つように震えた。」
- **三人称視点を統一する。** POVキャラの内面は描けるが、相手の内面は行動・表情・声から推測させる
- **環境を消さない。** シーン中にも世界を維持する。砂埃、ガソリンの匂い、遠くの銃声、壁のひび割れ、焚火の残り火、風の冷たさ。「ここがどんな場所か」を忘れさせない
- **設定値の直書き禁止。** 数値やスペック用語を描写に変換する
- **擬音は控えめ。** 環境音（風・焚火・エンジン・砂嵐）と混ぜて使う。擬音単独行は原則避ける

---

## 品質基準

- **「〜た。」3文連続禁止。** 文末バリエーションを強制する（体言止め、連用中止、現在進行形、倒置、省略・余韻）
- **感覚の内側に入る。** 受け手が何を感じているかを書く（温度・圧力・匂い・痺れ・嫌悪・快感・恐怖）
- **五感を散りばめる。** 汗の匂い、肌の熱、砂の擦れ、血の鉄臭さ、ネオンの化学的な甘さ
- **同じ部位・対象でも表現を変える。** 繰り返しを避け、場面ごとに語彙を使い分ける

---

## 濃淡コントロール

シーン執筆時に「濃度」を指定し、描写密度を制御する。

| 濃度 | 身体描写 | 心理描写 | 擬音 | 文字数目安 |
|---|---|---|---|---|
| **高** | 詳細。部位・感触・温度・反応を書く | PCの内面変化を追う | 最小限だが使用可 | 1500〜3000字 |
| **中** | 要点のみ。何が起きているかは伝えるが一挙手一投足は追わない | 心理が主体 | ほぼ不使用 | 500〜1500字 |
| **低** | 暗転 or 極度の圧縮 | 事前と事後の心理のみ | 不使用 | 200〜500字 |

### 濃度選択の基準

- PCの主体性が高い（自発的に望んだ）→ 濃度を上げられる
- 享楽イベント → 高を標準とする
- 取引的・制度的 → 文脈次第で高まで許容
- 変異獣・ブルードマザー → 中〜高を許容（ホラーとしての生理的詳細がトーンの核）
- 暴力隣接 → 濃度制限なし。感情軸とゲーム設計上の意図で決める
- 目撃シーン → 中が上限（PC視線が主役）
- 一般イベント（非性的）→ 中を標準。戦闘・交渉・探索は内容に応じて調整

---

## シーン構造テンプレート

### 標準3幕（享楽・同行者シーン）

| 幕 | 内容 | 重点 |
|---|---|---|
| 第1幕：文脈 | なぜこの場でこの相手と。PCの心理状態。環境描写 | 世界観への接地 |
| 第2幕：行為 | 濃度タグに従った身体描写。PCの内面変化 | 快感 and/or 心理の動き |
| 第3幕：余韻 | 事後の身体・心理。世界への帰還。ゲーム的結果との接続 | 感情の着地 |

### 簡略2幕（取引的・制度的シーン）

| 幕 | 内容 |
|---|---|
| 第1幕 | 状況と選択。PCが応じる/拒否する判断。応じた場合の心理 |
| 第2幕 | 行為＋余韻。行為は圧縮。事後の心理反応に重点 |

### 暗転型（低濃度シーン）

行為そのものは描写せず、直前と事後のみで構成する。暗転の「間」で何があったかを読者に想像させる。

### ネオン使用シーン（通常の1.2倍の文字数）

通常の構造に「知覚変容レイヤー」を追加する。

- 軽度使用：感覚鋭敏化。五感描写が通常より鮮烈に
- 重度使用：感覚の遠距離化。「自分の身体が他人のもの」感覚
- 過剰使用：記憶断絶。テキスト上で「空白」を表現。場面が飛ぶ。PCが自分の身体の痕跡から推測する

### 目撃シーン

- **PCの視線が主役。** 行為そのものより、PCがそれをどう見ているかを書く
- 視線の動き・呼吸の変化・手の位置（武器に伸びるか、握りしめるか、力が抜けるか）で内面を示す
- **介入選択肢がある場合：** 介入する/しないの両方に心理的代償を描く
- **介入不可の場合：** PCの無力さを強調するが、その光景が以後の選択に影響する種を蒔く
- **被害NPCの描写：** 被害者を「かわいそうな存在」に還元しない。恐怖・怒り・諦め・無感覚など個体として反応を持たせる

### 一般イベント（非性的）

| 幕 | 内容 |
|---|---|
| 状況提示 | 環境描写＋何が起きているか。PCの五感を通して伝える |
| 選択と行動 | PCが判断し動く。選択肢がある場合はここで分岐 |
| 結果と余韻 | 行動の結果。世界やキャラへの影響を1〜2文で着地させる |

## 性的シーンの文体追加ルール

- **♡は女性キャラの台詞にのみ使用可。** 男性キャラの台詞には使用しない
- **♡の密度は快感強度に連動する：**
  - 通常の快感：1台詞0〜1個
  - 強い快感（絶頂直前・弱点刺激）：1台詞1〜2個
  - 絶頂・ネオン使用中：1台詞2〜3個（上限3個）

### リリンのダンジョンスキルとの差分

| 項目 | リリンのダンジョン | Too Fast to Die |
|---|---|---|
| 視点 | エミ一人称寄り | 三人称統一 |
| ♡ | 1台詞最大3個 | 不使用 |
| 擬音密度 | 地の文3-4文に1回 | 必要最小限 |
| 嬌声テンプレート | 5段階のサンプル台詞 | 心理・身体の状態変化として定義 |
| 環境描写 | シーン内では最小限 | 常に世界と接地 |
| トーン | 快楽特化 | 世界観統合・濃淡可変 |

## 変異獣シーンの執筆規則

### ブルードマザー

- **生態的動機：** 繁殖。本能のみ。意志・感情・知性はない
- **PCの内面：** 恐怖・嫌悪が主体。快楽は「身体の裏切り」として処理する
- **環境描写：** 巣の内部。粘液・暗闘・体温・湿度。閉鎖空間の圧迫感
- **脱出の保証：** ゲーム的に脱出手段は必ず存在する（同行者救援、固有スキル等）
- **調教師（ミーシャ）の特例：** 変異獣の生態への理解がある。恐怖より観察が先に来る。巣からの脱出成功率が高い

### その他変異獣

種ごとに以下3項目を差分定義する。

| 項目 | 定義内容 |
|---|---|
| 身体構造の特異点 | 通常の動物/人間との差異。変異による形態変化 |
| 行為の動機 | 繁殖・捕食・支配・好奇心・本能暴走 |
| PCの反応パターン | 恐怖・嫌悪・観察・抵抗・屈服・逆転 |

### 変異獣シーンの描写原則

- 変異獣の行為は「自然現象」として書く。善悪の判断を地の文に入れない
- PCの抵抗と快感の葛藤を軸にする。身体が「裏切る」過程を丁寧に
- 調教師ルートでは、獣との関係が「信頼・共生・訓練の延長」としての独自の質感を持つ。単純な獣姦として処理しない

## 暴力×性の描写原則

- **暴力は世界観の反映として描く。** ポストアポカリプスでは暴力は日常
- **PC加害描写も許容。** カルマシステムとの連動を必ず意識する
- **強制シーンの濃度制限なし。** 感情軸とゲーム設計上の意図で決める

### パターン分類

| パターン | 発生場面 | PC立場 | 濃度 |

**ヴェスパ（受け）：**

## 合意/非合意による描写分岐

### 合意シーンの原則

- PCの選択として描く。「なぜこの相手と、なぜ今」の文脈を第1幕で確立
- 快感描写に重点を置ける。心理的障壁が低い分、身体描写の密度を上げられる
- 事後は関係性の変化を書く

### 非合意シーン（PCが襲われる）の原則

- **脱出/逆転の余地を維持。** 完全な絶望で終わらせない
- **心理描写を身体描写より優先。** 何をされているかより、PCが何を感じ何を考えているか
- **身体の「裏切り」：** 快感が発生する場合、PCの意志に反していることを明示
- **PC別反応：**
  - アタルパ：信仰に縋る。「これは試練だ」と自分に言い聞かせる
  - ヴェスパ：冷静に脱出を計算する。コカトリスで慣れている自分への嫌悪
  - ウェズリー：静かに耐える。30年間で何度もあった。感情が摩耗している
  - ミーシャ：硬直→フラッシュバック。虎がいれば虎が暴れる
  - 覇者：怒り。力で返す。非合意の状態が最も短いPC
  - ホタル：パニックにならない。身体を弛緩させて受け流す（歓楽街の護身術）。ただしネオンが切れている状態では化学的鎧がなく、恐怖が鮮明になる

### 非合意シーン（PCが襲う）の原則

- **カルマ大幅減。** ゲーム的結果を必ず意識
- **PCの内面を書くか書かないかが選択。** 書かない（無感覚）は低カルマPCの表現
- **被害NPCを個体化。** 恐怖・怒り・諦め・無感覚のいずれかを持たせる
- **PC別の加害パターン：**
  - アタルパ：狂信状態なら「儀式」として正当化。懐疑以下では発生しにくい（選択肢自体が消える）
  - ヴェスパ：コカトリス時代の行動パターンへの回帰。ヒート上昇と連動
  - ウェズリー：最も発生しにくいPC。発生した場合、テクニックの冷徹さが恐怖を生む
  - ミーシャ：人間に対しては発生しない。虎を使った制圧はあり得る
  - 覇者：力の延長。「壊してしまう」パターン
  - ホタル：歓楽街での「営業」パターンへの回帰。色仕掛けから始まり、主導権を握ったまま搾取する。加害の自覚がない——これが「仕事」だった。カルマ減少は小さいが、それ自体が彼女の壊れ方の証
